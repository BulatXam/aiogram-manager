# <p align="center"> Aiogram Manager</p>
Project manager and generator for Aiogram



___

## Installation

``` python
pip install aiogram-manager
```
___

## How to use

``` bash
aiogram startproject [projectname]
```

``` bash
cd codingbot
```

``` bash
aiogram startapp [appname]
```

___

## Utils options

``` bash
aiogram --help
```
